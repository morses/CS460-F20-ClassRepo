Form Examples

A good place to start is: MDN Web forms -- Working with user data. This version has a bare minimum of Bootstrap styling on the first few forms.

You need to "run" these with a web server rather than loading the file via file:// URI. A good choice is to install Node.js via NPM and then install the simple "http-server" package:

npm install http-server -g

Run by cd'ing to this directory from the command line then run:

http-server

You can also just wait until we put this in an ASP.NET project.
